tomosphero.loss

Optimizer loss functions

The classes in this module serve as loss functions for the iterative reconstruction methods in retrieval.py.

The class should be initialized by the user and then passed to the reconstruction method which will call .compute() on the loss function and try to minimize the weighted sum of all provided losses.

Weights are stored internally in the lam parameter, which the user may set by multiplying the initialized loss object with a float or by providing a lam kwarg on initialization.

  2
  3The classes in this module serve as loss functions for the iterative reconstruction methods in `retrieval.py`.
  4
  5The class should be initialized by the user and then passed to the reconstruction method which will call `.compute()`
  6on the loss function and try to minimize the weighted sum of all provided losses.
  7
  8Weights are stored internally in the `lam` parameter, which the user may set by multiplying the

 14class Loss:
 15    """Base class for tomographic retrieval loss function terms
 16
 17    Subclass this class and implement `compute(…)` to create new loss function terms.
 18    Instantiating Loss objects may be multiplied by a float to set weight (see Usage).
 19
 20    Args:
 21        projection_mask (tensor): column densities to mask out when computing loss
 22        obj_mask (tensor, optional): voxels to mask out when computing loss
 23        lam (float, optional): loss function scaling (default 1)
 24        use_grad (bool, optional): whether this loss function's gradient needs to be
 25            used in optimization (default True)
 26
 27    Usage:
 28    ``` python
 29        gd(..., loss_fns=[5 * MyLoss(), 3 * MyLoss2()], ...)
 30    ```
 31
 32    Attributes:
 33        kind (str): Category of loss - one of either 'fidelity', 'regularizer', or 'oracle'.
 34            - fidelity - term is data-fidelity term
 35            - regularizer - term is a regularizer
 36            - oracle - term is for logging purposes, not used in loss function
 37            This property is used by `loss_plot(…)` for display purposes.
 38            Function `gd(…)` also uses this property when generating loss term summary
 39            in the TQDM status bar.
 40        lam (float): weight hyperparameter for this term
 41
 42    """
 43
 44    kind = 'regularizer'
 45
 46    def __init__(
 47            self, *args, projection_mask=1, obj_mask=1, lam=1,
 48            use_grad=True, **kwargs
 49        ):
 50        """@private"""
 51        self.projection_mask = projection_mask
 52        self.obj_mask = obj_mask
 53        self.lam = lam
 54        self.use_grad = use_grad
 55
 56    def compute(self, f, y, d, c):
 57        """Compute loss
 58
 59        Args:
 60            f (Operator): forward function. density→projections
 61            y (tensor): measurements.  shape must match `projection_mask`
 62            d (tensor): density to pass through forward function.
 63                shape must match `obj_mask`
 64            c (tensor): coefficients of shape model.coeffs_shape
 65
 66        Returns:
 67            loss (float)
 68        """
 69        raise NotImplemented
 70
 71    def __call__(self, f, y, d, c):
 72        """Compute loss, incorporating loss weight and whether pytorch grad is needed
 73
 74        Args:
 75            f (Operator): forward function. density→projections
 76            y (tensor): measurements.  shape must match `projection_mask`
 77            d (tensor): density to pass through forward function.
 78                shape must match `obj_mask`
 79            c (tensor): coefficients of shape model.coeffs_shape
 80
 81        Returns:
 82            loss (float or None)
 83        """
 84        if self.use_grad:
 85            result = self.compute(f, y, d, c)
 86        else:
 87            with t.no_grad():
 88                result = self.compute(f, y, d, c)
 89        return None if result is None else self.lam * result
 90
 91    def __mul__(self, other):
 92        """Allow multiplying Loss object with scalar hyperparameter"""
 93        self.lam = other
 94        return self
 95
 96    def __rmul__(self, other):
 97        """Allow multiplying Loss object with scalar hyperparameter"""
 98        return self.__mul__(other)
 99
100    def __repr__(self):
101        return f'{self.lam:.0e} * {type(self).__name__}'
102        # return f'{type(self).__name__}'

116class SquareRelLoss(Loss):
117    """Loss as mean percent error"""
118
119    kind = 'fidelity'
120
121    def compute(self, f, y, d, c):
122        """"""
123        obs = f(d * self.obj_mask)
124
125        # rel_err = (y - obs) / y
126        # rel_err = rel_err.nan_to_num() * self.projection_mask
127
128        zero_mask = (y != 0)
129        rel_err = t.zeros_like(y)
130        rel_err[zero_mask] = (y - obs)[zero_mask] / y[zero_mask]
131
132        return t.mean((self.projection_mask * rel_err)**2)

215            d (tensor): candidate reconstructed density
216            c (tensor): coefficients for model.  low-rank representation of

218

220            loss (float)

222        # only perform computation every `interval` calls to save on compute time
223        # otherwise, return NaN
224        if self.n % self.interval == 0:
225            rel_err = t.abs(self.truth - d) / self.truth
226            if self.mode == 'mean':
227                result = rel_err[self.truth_cart > 25].mean()

231                raise ValueError("Invalid mode")

235        self.n += 1
236        return result
